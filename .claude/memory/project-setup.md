code-atlas（代码图谱）——一个让不写代码的 PM 也能看懂项目全貌的代码分析与可视化工具。

**核心功能**：
- 分析引擎：扫描代码库，提取模块、函数、依赖关系（支持多语言：TS/Python/Go/Java/Kotlin）
- 前端可视化：模块地图（三层：有什么 → 怎么连 → 对应什么业务）+ 后续依赖链路/风险标注/时间线
- CLI 接口：AI 开发时加载项目知识库作为上下文
- POST 标注系统：按模板持续补充/修正模块关键信息

**部署方式**：Docker，部署在 NAS 上，支持多项目管理

**搭建日期**：2026-04-27
